# 恐龍世界 · Dinosaur World

寫實的白堊紀谷地互動教學模擬器 —— 遠觀整座谷地的生態,或以第一人稱走進恐龍身邊近距離觀察,附完整中文教學資訊。純前端、零依賴、零建置。

## 特色

- **兩種視角**:遠觀(俯瞰谷地)/ 漫遊(第一人稱 WASD + 滑鼠環顧)
- **八種恐龍**:暴龍、三角龍、腕龍、伶盜龍、劍龍、副櫛龍、甲龍、風神翼龍;各有程序化 3D 模型與教學資料卡
- **時間系統**:滑桿控制一天的時刻,天色與光影即時變化
- **教學功能**:物種圖鑑、資料卡(尺寸/年代/化石產地/趣聞/與人類比例)、生態導覽 8 站
- **程序化生成**:地形、河流、森林、貼圖、天空全部即時算出,整個專案沒有任何圖片檔

## 執行

```
node tools/serve.mjs 8124
# 開 http://localhost:8124/
```

## 操作

| 動作 | 遠觀 | 漫遊 |
|---|---|---|
| 旋轉/環顧 | 拖曳 | 滑鼠 |
| 縮放 | 滾輪 | — |
| 移動 | — | W A S D(Shift 奔跑) |
| 聚焦恐龍 | 點擊恐龍或標籤 | — |
| 切換視角 | 按 `2` 進漫遊 | 按 `1` 回遠觀 |
| 暫停時間 | 空白鍵 | — |
